Appendix B Assembler Directives

It is considered good practice to prepend a period to a directive when invoking it (to differentiate it from identifiers), but this is not a strict requirement.

The following is an alphabetical listing of the directives supported by the assembler and the arguments they may take. Arguments enclosed in square brackets [ ] are optional.

align identifier [, identifier-2, ...] boundary
Specifies alignment constraints for a list of data variables.
asc (Alias for char)
byte (Alias for db)
char expression [, expression, ...]
Define (array of) character transformed by custom character map
charmap "filename"
Set custom character map
codeseg, code
Switch to code segment
dataseg, data [zeropage]
Switch to data segment
db expression [, expression, ...]
Define (array of) byte
dd expression [, expression, ...]
Define (array of) doubleword
define identifier [expression]
See equ directive
dsb expression
Define storage of bytes
dsd expression
Define storage of doublewords
dsw expression
Define storage of words
dw expression [, expression, ...]
Define (array of) word
dword (Alias for dd)
elif
Used in conjunction with if
else
Used in conjunction with if, ifdef or ifndef
endif
Ends a statement block preceded by an if
end
Ends the assembly unit
ende
Ends an enum definition
endm
Ends a macro definition
endp
Ends a procedure definition
ends
Ends a structure or union definition
enum identifier
Begins an enum definition
extrn identifier [, identifier, ...] : type
Flag identifier(s) as external (imported) of type type
identifier equ expression
Define equate
if expression
Assemble the following statement block only if expression evaluates to non-zero
ifdef identifier
Assemble the following statement block only if identifier is defined
ifndef identifier
Assemble the following statement block only if identifier is not defined
incbin "filename"
Include contents of filename as binary data
include (Alias for incsrc)
incsrc "filename"
Include contents of filename as assembler statements
label identifier [= address] [ : type]
Defines a global label
macro identifier [identifier, ...]
Begins a macro definition
message expression
Prints a message to stdout during assembly
pad expression (Alias for dsb)
proc identifier
Begins a procedure definition
public identifier [, identifier, ...]
Flag identifier(s) as public (exported)
record identifier identifier:width [, identifier:width, ...]
Defines a record consisting of bitfields.
rept count
Begins an anonymous macro to be repeated count times
struc identifier
Begins a structure definition
type identifier [expression, ...]
Define data of user-defined type identifier
union identifier
Begins a union definition
while expression
Begins an anonymous macro to be repeated while expression is true (non-zero)
word (Alias for dw)
